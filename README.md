## About This Starter Kit

This React starter kit provides a robust, modern starting point for building Laravel applications with a React frontend
using [Inertia](https://inertiajs.com).

Inertia allows you to build modern, single-page React applications using classic server-side routing and controllers.
This lets you enjoy the frontend power of React combined with the incredible backend productivity of Laravel and
lightning-fast Vite compilation.

### Tech Stack

- **Backend:** Laravel –
  featuring [simple routing](https://laravel.com/docs/routing), [dependency injection](https://laravel.com/docs/container), [Eloquent ORM](https://laravel.com/docs/eloquent), [migrations](https://laravel.com/docs/migrations), [queues](https://laravel.com/docs/queues),
  and [real-time events](https://laravel.com/docs/broadcasting)
- **Frontend:** React 19, TypeScript, Tailwind CSS
- **Components:** [shadcn/ui](https://ui.shadcn.com) and [base-ui](https://base-ui.com/)
